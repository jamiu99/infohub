# infohub v0.1.2

发布日期：2026-07-13

这是一个公众号登录修复版本。

## 修复

- 微信公众平台优先显示账号或“微信快捷登录”时，自动切换到传统二维码。
- 避免 Electron 中本地网络权限状态未完成导致扫码区域空白。
- 登录窗口明确提示只用手机扫码，不要输入账号、密码或验证码。
- 登录 Session 默认拒绝设备权限，只允许 `https://mp.weixin.qq.com` 顶层导航，并阻止页面弹窗。
- 新增二维码切换和登录 URL 白名单测试，核心测试增至 37 项。

## 升级说明

- 已安装 `v0.1.1` 的用户会收到自动更新；也可以从 GitHub Release 手动下载安装。
- 本版本仍会在扫码成功后把 Cookie/token 加密保存在本机，这是公众号采集接口所需的会话凭据。
- Windows 安装包尚未代码签名，SmartScreen 可能显示“未知发行者”。

## 待验收

- 需要在真实 Windows 环境确认传统二维码显示、扫码确认、关闭窗口保存账号和首次公众号采集完整通过。
