# infohub v0.3.1

发布日期：2026-07-15

这是一次 Windows 品牌修正版，不改变采集、存储、团队同步协议或数据格式，也不增加任何 AI 直接集成。

## Windows 图标

- 将项目 logo 正式纳入仓库：`resources/branding/infohub-logo-concept-v1.png` 保留概念源图，`resources/branding/infohub-icon-v1.png` 是用于打包的紧凑透明图标。
- Windows 应用 EXE、任务栏/快捷方式以及 NSIS 安装和卸载界面统一使用项目 logo，不再使用 Electron 默认图标。
- electron-builder 会从 1254×1254 RGBA PNG 生成 Windows ICO；已用与发布流程相同的图标转换器验证输入可正常转换。
- bundle 门禁新增品牌检查：打包图必须存在、为至少 512×512 的方形 RGBA PNG，打包配置必须明确引用该 logo。

## 升级与兼容

- 可以从 `v0.3.0` 通过 App 内“检查更新”升级，或直接运行 `v0.3.1` 安装包覆盖安装。
- 本版本没有数据库迁移，用户数据、登录状态、设置、布局和团队配置均沿用现有目录。
- 团队同步继续使用 `v0.3.0` 引入的破坏性 `/api/v2` 协议，不支持旧服务端或旧桌面端混用。

## 当前限制

- Windows 安装包仍未代码签名，SmartScreen 可能显示“未知发行者”。
- 自动化可验证图标配置和转换结果；Windows 资源管理器、任务栏及安装向导中的最终视觉效果仍需在真实 Windows 安装后人工确认。
