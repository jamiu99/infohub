# infohub v0.1.1

发布日期：2026-07-12

这是 infohub 完成产品边界收敛后的第一个版本：它是本地信息采集、文件归档、SQLite 索引和快速看板，不再直接集成任何 AI 能力。

## 主要变化

- 永久移除 Claude/Codex CLI、内置 Skill、摘要和简报逻辑；AI 与普通脚本一样，只通过文件系统或 SQLite 读取数据。
- 自动生成 `data/INFOHUB_DATA.md`，并补齐文件布局、稳定字段、索引表和读取约束。
- Article 文件新增持久化 `externalId`；状态同时写入文件和索引，SQLite 可从文件完整重建。
- 增加 schema v2 迁移、原子写入、路径防逃逸、外部文件变更同步和去重表重建。
- 增加 DOMPurify、CSP、外链白名单和 renderer sandbox，降低不受信任内容风险。
- 修复 `start.sh` 对残留 tmux 会话的识别和自动清理。
- 发布工作流现在强制校验 tag/版本一致，并在 Windows 打包前运行类型检查和核心测试。

## 安装与升级

- Windows 安装包为 x64 NSIS 当前用户安装，不要求管理员权限，可选择安装目录。
- App 会在打包环境启动后自动检查 GitHub Release；发现新版本后后台下载，并提示重启安装。
- 本版本尚未代码签名，Windows SmartScreen 可能显示“未知发行者”。
- 从旧版本升级时，用户目录里遗留的 `.claude/skills/`、`briefings/` 不会自动删除，但 infohub 已完全忽略它们。
