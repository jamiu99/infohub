# infohub v0.1.4

发布日期：2026-07-13

这是传统桌面更新体验版本。

## 更新流程

- 启动后自动检查更新，但发现新版时先询问用户，不再直接后台下载。
- 可从“帮助 → 检查更新”或左栏“检查更新”按钮手动触发。
- 用户确认后才下载，界面提示条和 Windows 任务栏显示进度。
- 下载完成后询问“立即重启更新 / 稍后”；选择稍后会在退出时安装。
- 手动检查会明确显示“当前已是最新版本”或错误信息。
- 核心确认对话框和帮助菜单运行在 Electron main，不依赖 Vue/preload，可作为故障恢复入口。

## 验证

- 42 项核心测试通过，其中 5 项覆盖更新状态机。
- sandbox preload 打包契约和真实 Electron preload/IPC smoke test 继续作为 CI 门禁。

## 升级路径

- 旧 `v0.1.1` 的 preload 已损坏，需要先手动安装 `v0.1.3` 恢复桌面桥接。
- `v0.1.3` 启动后会发现 `v0.1.4`，可用本版本新增流程完成确认、下载和重启安装。
- Windows 安装包尚未代码签名，SmartScreen 可能显示“未知发行者”。
